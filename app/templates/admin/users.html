User Management Guide
System Admin
  • Can manage all users system-wide
  • Can view and access all projects
  • Can create/delete projects without restrictions
  • Can grant/revoke admin privileges
Regular User
  • Can only access projects they're assigned to
  • Cannot manage other users system-wide
  • Can be Owner/Maintainer/Reader in projects
  • Can create new projects (becomes owner)
